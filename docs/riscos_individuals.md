# Riesgos laborales individuales

A continuación se describen los riesgos detectados durante el trabajo frente al ordenador y posibles medidas de mejora:

| Riesgo | Categoría | Mejora propuesta |
|--------|-----------|----------------|
| Fatiga visual por mirar la pantalla muchas horas | Físico | Hacer pausas cada 45-60 min, ajustar la luz y contraste de la pantalla, aplicar la regla 20-20-20 |
| Dolor de espalda por mala postura | Físico | Ajustar silla y escritorio, mantener postura recta, estiramientos cada hora, usar cojín lumbar |
| Estrés por deadlines y presión | Psicosocial | Planificar tareas con antelación, dividir el trabajo en microtareas, tomar pausas cortas, comunicar bloqueos |
| Fraude en transacciones digitales | Financiero | Implementar sistemas de detección automática de tarjetas robadas y bloqueos geográficos |
| Robo o filtración de datos/código | Técnico | Usar autenticación de dos factores (2FA) en todas las cuentas de programador |
| Error crítico en script de entrega | Operativo | Crear un sistema de alertas que avise al equipo si se entregan más de X códigos por minuto |
| Dependencia de un solo proveedor | Estratégico | Establecer contacto con al menos dos proveedores alternativos para casos de emergencia |
| Sedentarismo y falta de ergonomía | Físico | Alternar periodos de trabajo de pie si es posible y realizar ejercicios de movilidad articular |
